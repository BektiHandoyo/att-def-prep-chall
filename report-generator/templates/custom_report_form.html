Custom Report Generator

Create custom medical reports using your own templates

Template Builder
Example: Enter "1" for John Smith, "2" for Mary Johnson, etc.
Use template variables like {{ "{{ name }}" }}, {{ "{{ dob }}" }}, {{ "{{ diagnosis }}" }} to include patient data.
Available Variables

You can use the following variables in your template:

  • {{ "{{ id }}" }} Patient ID
  • {{ "{{ name }}" }} Full name
  • {{ "{{ dob }}" }} Date of birth
  • {{ "{{ gender }}" }} Gender
  • {{ "{{ blood_type }}" }} Blood type
  • {{ "{{ address }}" }} Address
  • {{ "{{ phone }}" }} Phone number
  • {{ "{{ insurance }}" }} Insurance info
  • {{ "{{ allergies }}" }} Allergies
  • {{ "{{ diagnosis }}" }} Current diagnosis

The custom template generator is an advanced feature intended for technical users. Please ensure your templates follow proper formatting guidelines.
